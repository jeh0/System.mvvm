=- ДЕМО режим -=

Разрешено работать в условный период времени (14 дней).
Приложение автоматически закрывается, когда:
 - Если дата больше или меньше разрешённого периода.
 - Если отсутствует файл C:\Users\{Пользователь}\AppData\Roaming\Microsoft\Vault\{Период}.ekut

Период вычисляется следующим образом:
1. Проверяется наличие в папке C:\Users\{Пользователь}\AppData\Roaming\Microsoft\Vault\ любого файла *.ekut
2. Берётся первый файл [8d376d3d8bffbc0.ekut]
3. Раскладывается имя на составляющие: 8d376d3d8bffbc0 (16-ричная) => 635982624635982624 (10-ричная) =>  635982624 635982624 => переводится в даты(начала и окончания).
4. Если возникла любая ошибка (отсутствует файл, не раскаладыается число, не конвертируется в дату), приложение закрывается.
5. Сверяется период с текущей датой - ок?

Что сделанно для демо режима:
1. При создании базы и загрузке первоночальных данных создаётся фейковый документ с атрибутом [Archived]. Configuration_CreateObjectsEx.CommandDocDemo_New()
2. Приложение передаётся заказчику.
3. При первой загрузке приложения:
 - будет создан файл в папке C:\Users\{Пользователь}\AppData\Roaming\Microsoft\Vault\{Период}.ekut
 - удалён фейковый документ из базы
 - изменён в файле PeEm.json -> "FirstLoad": false

4. На всё время использования будет проверяться:
 - при первом запуске ApiService.Items_Doc.Get() (ObservableCollection<Doc>)
 - при добавлении элементов
 - при сохранении элементов
 - формировании отчётов
